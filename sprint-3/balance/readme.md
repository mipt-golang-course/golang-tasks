## balance

Дана строка s, содержащая символы (, ), {, }, [, ] и символы латинского алфавита. Нужно определить, является ли строка s правильной.
Строка считается правильной, если:
Каждая открытая скобка имеет соответствующую закрывающую скобку того же типа.
Скобки закрываются в правильном порядке.
Символы латинского алфавита не влияют на правильность строки.

Подсказка: структура данных стек тут может прийтись как нельзя кстати

Пример 1:
Ввод: s = "Algo[[(C)od]]{e}the({Best}{[Pla]t}f)orm"
Вывод: true

Пример 2:
Ввод: s = "[]()))"
Вывод: false

Пример 3:
Ввод: s = "((()"
Вывод: false
