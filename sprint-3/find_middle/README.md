## find-middle

Дан односвязный список неизвестной длины. Необходимо найти его средний узел, используя алгоритм с двумя указателями.

Требования:
Алгоритм должен работать за O(n) времени.
Алгоритм должен использовать O(1) дополнительной памяти.
Если элементов четное количество, то возвращаем второй элемент
### Проверка решения

Для запуска тестов нужно выполнить следующую команду:

```
go test -v ./sprint-3/find-middle/...
```
