## merge_lists

Даны головы двух отсортированных односвязных списков — list1 и list2.
Необходимо объединить эти два списка в один отсортированный список.
Новый список должен быть построен путём «сшивания» узлов исходных списков (без создания новых узлов, если это возможно).
Нужно вернуть голову объединённого отсортированного списка.

### Проверка решения

Для запуска тестов нужно выполнить следующую команду:

